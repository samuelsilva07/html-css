Principais formatações

Negrito/Destaque

O termo em destaque está em negrito com a tag <b> (não semântica)

O termo em destaque está em negrito com a tag semâtica <strong> (negrito semântico)

Itálico/Ênfase

Na frase temos um termo em itálico com a tag <i> (não semântica)

Agora usaremos nesse trecho a tsg semântica <em> (itálico semântico)

Teste de envelopamento no VS Code

Texto marcado

Pode-se criar um texto marcado com a tag <mark>

Tamanhos de texto

Temos textos de tamanho grande e de tamanho pequeno (utilizada em final de contratos - tag <small>)

(A tag <big> é obsoleta, deve-se aumentar o tamanho no CSS)

Texto excluído

É possível marcar um texto como excluído no HTML com a tag <del>

Texto inserido

Existe também o texto inserido, feito através da tag <ins>

Existe também a tag <u>, no entanto ela não é semântica

Texto sobrescrito

Para inserir textos do tipo x2+3, usa-se a tag<sup>

Texto subscrito

Para inserir textos do tipo H20, usa-se a tag <sub>
